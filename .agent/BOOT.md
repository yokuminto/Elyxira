# Elyxira

- **版本**: 2.0.2
- **类型**: 静态 SPA（GitHub Pages 部署）
- **目的**: 开源医学刷题 + 绯想击破游戏化模式 + AI 知识点精讲
- **技术栈**: TypeScript + Vue 3 (Composition API `<script setup>`) + Vite 6 + Vue Router (hash)
- **关键依赖**: markdown-it, vue-toastification, vue-router, yjs, y-webrtc, y-indexeddb, qrcode, FontAwesome 6
- **仓库**: `yokuminto/Elyxira`（源码）→ CI 部署到 `elyxira.github.io`
- **数据仓库**: `elyxira/elyxira-data`（共享笔记/标签/题库）
